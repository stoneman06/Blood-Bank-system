# Blood-Bank-system
Blood Bank and Management system built using PHP and MYSQL
# Blood-Bank-&-Donation-Management-System


## Installation

1. Install XAMPP or WAMPP.

2. Open XAMPP Control panal and start [apache] and [mysql] .

3. Download project 
     
4. Extract files in C:\xampp\htdocs.

5. Open link localhost/phpmyadmin

6. Click on new at side navbar.

7. Give a database name as (blood_bank_database) hit on create button.

8. After creating database name click on import.

9. Browse the file in directory[BDMS/sql/blood_bank_database.sql].

10. After importing successfully.

11. Open any browser and type http://localhost/BDMS/index.php to open User layout.
